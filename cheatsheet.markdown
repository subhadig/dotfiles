## Reload .vimrc from within vim
- :so $MYVIMRC

## Searching within the opened file
- :vimgrep pattern %
- :copen
- :cn
- :cp
- :ccl

## Spliting
- :split
- :vsplit

## Resizing
- :res +10
- :vertical resize +10

## Navigation
- Move half page down -> Ctrl + D
- Move half page up -> Ctrl + U
- Move view pane down -> Ctrl + E
- Move view pane up -> Ctrl + Y
